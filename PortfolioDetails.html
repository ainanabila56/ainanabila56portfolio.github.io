Reader's Haven

Create an online reading app named "Reader's Haven" that allows users to discover, read, and publish their stories. Users can browse through various genres, follow their favorite authors, and engage with a community of readers and writers.

  • Rating:
  • Technologies: Frontend: React.js (or Vue.js), HTML, CSS
  • Used Backend: Node.js with Express

  • Cloud Storage: Amazon S3
  • Version Control: Git
  • Completed: 10 Mac 2023

Craftify Marketplace

Craftify Marketplace is an online platform for artisans and craft enthusiasts to showcase, sell, and purchase handmade crafts. Users can browse through a diverse range of crafts, connect with sellers, and build a community around creativity.

  • Rating:
  • Technologies: Frontend: React.js, HTML, CSS
  • Used Backend: Django (Python framework)
  • Database: PostgreSQL
  • Payment Authentication: Stripe or PayPal
  • Authentication: OAuth (Google/Facebook login)
  • Cloud Storage: Amazon S3 (for storing craft images)
  • Version Control: Git
  • Completed: 6 Sept 2021

SmileSync Dental Booking

SmileSync is a dental booking app designed to simplify the process of scheduling dental appointments for users. It allows patients to view available appointment slots, and book appointments at their convenience. Dentists can manage their schedules, view patient details, and streamline their appointment workflows.

  • Rating:
  • Technologies: Frontend: React Native, Expo, HTML, CSS
  • Used Backend: Node.js with Express
  • Database: MongoDB
  • Authentication: JWT (JSON Web Tokens)
  • Version Control: Git
  • Completed: 16 June 2018
